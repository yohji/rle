# Ruby Language Extensions

`rle` is a set of extensions of core classes of Ruby.

## Installation

```
$ git clone https://github.com/yohji/rle.git
$ gem build rle.gemspec
$ sudo gem install ./rle-XXX.gem
```

## Usage

```ruby
require 'rle'
```
